# Привет, я - Женя. 

---

### Обо мне

3 года я являлся начальником отдела информации в типографии Константа. 
В мои обязанности входило администрирование <a href="https://konstanta-print.bitrix24.ru/" > портала Bitrix24 </a> (веб-версия), помощь в администрировании Asystem, ведение <a href="https://konstanta-print.ru/" > основного сайта компании </a> и <a href="https://vizitkitut.ru" > вспомогательных </a> . В списке обязанностей было:
 Автоматизация CRM системы, настройка роботов / триггеров на всех стадиях прохождения сделки/лида, привязка их к Бизнес-процессам.Создание более 70 Бизнес-процессов (от простеньких "Создать аккаунт Bitrix" до "Закупка и доставка материалов, которая длится до 3х месяцев"). 
<p>
В процессе работы с Bitrix24 заинтересовался программированием, начал с HTML + CSS + PHP + SQL, начал создавать сайты на Wordpress 
для закрепления теории практикой. Данный стак осваивал ~год, затем JS + TypeScript + Bootstrap + Tailwind.
Затем прошёл курсы <a href="https://www.sql-ex.ru/"> SQL </a> , получил сертификат, изучил фреймворки Laravel, Symphony, VueJS, познакомился с API, JSON, jQuery.
<br>

В 2023г. перешёл в МКК "АксФинанс". <br>
С начала 2023г. до ноября 2023г. был веб-разработчиком, затем стал ведущим веб-разработчиком. <br>
Моя деятельность в роли веб-разработчика:
Администрирование Bitrix24.
Поддержание/Развитие нашего сервиса-лидогенератора (внутренние и внешние API). Взаимодействие с контрагентами (помощь в интеграции, постбеки).
Создание корпоративных сайтов и лендингов, интеграция их с лидогенератором (в основном SPA приложения на Laravel/Lumen + VueJS3).
Разработка для телефонии Манго.
Разработка витрины данных для аналитиков.
Большой опыт работы по разработке и поддержке аналитических сервисов, например:
- Телефония (Манго) + выданные займы. Автоматизированные отчёты по расчёту ЗП колл-менеджерам, по качеству прозвона, по эффективности рекламных кампаний.
- Лидогенераторы онлайн-трафика + путь клиента + проверки нашими отделами + выдачи и постбеки. Разработан сервис сквозной аналитики с множеством отчётов. Автоматизация с помощью reddis и supervisor.
Интеграция с различными контрагентами (Манго, РоботМиа, SAAS).
Разработка для Bitrix24 (связывал Bitrix и 1С:УИТ , делал единое окно приёма и обработки заявок).
<br>
Моя деятельность в роли ведущего веб-разработчика:
Организация отдела веб-разработки из 3х сотрудников.
Личные результаты на данной должности:
1) Реализована продажа отказного трафика с 4 контрагентами по API в онлайне (личном кабинете).
2) Реализована продажа отказного трафика в оффлайне (перепродажа ФИО + телефона). > 2 млн лидов / мес.
3) Реализован сервис мониторинга своих систем (некий дубляж Zabbix, но кастомизированный под себя, с возможностью отслеживания перезапуска провальной очереди 1й кнопкой).
4) Реализован небольшой сервис мониторинга для маркетологов - лиды в разрезе месяцев / регионов / причин отказа.
5) Парсинг документов и информации с официального сайта на лендинги-визити (которые необходимы по ФЗ, т.к. у нас много юр.лиц). Раньше этим занимался сотрудник, тратя на это 20 часов+ / мес.

В 2024 прошёл курсы по CMS Wordpress (создание плагинов), парсинг на php (регулярки / Didom), парсинг на python (регулярки / BeatifulSoup4).


На <a href="https://crassh.ru">моём сайте</a> представлены мои проекты, в данный момент завершено 13 шт. Git почти не оформлен, мало что сюда заливал (самые первые версии сайтов).
</p>
